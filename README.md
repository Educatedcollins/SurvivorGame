# Survival Blocks

A 2D top-down survival game built in Unity 6 as a week-long learning project.

## 🎮 Play in Browser
[Play on itch.io](https://educatedcollins.itch.io/survival-blocks)

## 📖 About
Survive as long as possible against endless waves of enemy blocks that hunt 
you across the arena. Use screen wrapping to escape tight situations and 
watch for warning indicators before enemies spawn!

## 🕹️ Controls
- **WASD** — Move
- **ESC** — Pause (windowed mode)
- **Any key** — Return to menu after Game Over

## ✨ Features
- Endless survival gameplay with increasing difficulty
- Screen wrapping movement system
- Enemy spawn warning indicators
- Top 10 local leaderboard
- Full audio system with Master, Music and SFX volume controls
- Pause menu with in-game settings
- Particle effects and visual polish
- Blue player glow effect using Unity URP Light 2D

## 🛠️ Built With
- Unity 6 (Universal Render Pipeline)
- C#
- AI-assisted development via Claude (Anthropic)

## 📚 What I Learned
