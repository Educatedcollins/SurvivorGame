# Survival Blocks

A 2D top-down survival game built in Unity 6 as a learning project.

## Play in Browser
[Play on itch.io](https://educatedcollins.itch.io/survival-blocks)

## How to Play
- **WASD** to move
- Survive as long as possible against endless waves of enemy blocks
- Enemies spawn with a warning indicator before appearing
- **ESC** to pause (windowed mode)
- Press any key to return to menu after Game Over

## Features
- Top 10 local leaderboard
- Volume controls (Master, Music, SFX)
- Screen wrapping movement
- Particle effects and visual polish
- Pause menu with settings

## Built With
- Unity 6 (URP)
- C#
- AI-assisted development via Claude (Anthropic)
